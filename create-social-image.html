📸 How to create your social media image:

🎯 Method 1: Download Button (Try this first!)

Click the button above to automatically download the image as PNG.

🎯 Method 2: Screenshot (If download doesn't work)

  1. Press Windows + Shift + S (Snipping Tool)
  2. Select just the gradient image area above
  3. Save as social-image.png in your static folder

🎯 Method 3: Browser Developer Tools

  1. Right-click on the gradient image → "Inspect Element"
  2. Find <div class="social-image"> in the code
  3. Right-click that element → "Capture node screenshot"
  4. Save as social-image.png

Alternative: Use a tool like Canva, Figma, or Photoshop to create a 1200x630px image with your logo and text.
